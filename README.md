# Founder IP Prompt Library｜创始人IP提示词库

这是一个面向 B2B 企业创始人的 AI 提示词库。

目标不是让老板每天痛苦想选题、写文案、拍视频，而是把创始人的行业认知、客户问题、销售话术和案例经验，变成一套可复用的内容生产工作流。

---

## 适合谁使用

- 想做创始人IP但不知道讲什么的老板
- 高客单 B2B 企业
- 制造业、咨询、律所、财税、教育、移民等专业服务机构
- 需要用短视频建立信任的销售团队
- 想用 AI 提高内容生产效率的企业

---

## 提示词目录

### 01 账号诊断

用于判断账号定位、目标客户、内容方向和转化路径。

### 02 选题生成

用于从行业痛点、客户问题、成交阻力中生成短视频选题。

### 03 口播脚本

用于生成适合创始人口播的短视频文案。

### 04 客户案例提炼

用于把真实客户故事转化成可传播的案例内容。

### 05 私域转化话术

用于从短视频线索承接到微信、社群和成交。

### 06 数字人视频脚本

用于生成适合真人数字人批量生产的视频脚本。

---

## 使用原则

1. 先诊断，再生成
2. 先明确客户，再设计内容
3. 先解决信任，再追求流量
4. 先结构化经验，再使用 AI 放大
5. 先做最小闭环，再持续迭代

---

## 核心判断

创始人IP不是网红工程，而是信任工程。  
提示词不是咒语，而是业务结构的外化。
